# Determine-ex

A simple C++ program that determines:
- Whether the input is a numeric value
- Whether it contains operators (+, -, *, /, %, =)
- Whether it’s a single-line or multi-line comment

## Files
- `main.cpp` – main source code
- `lab1task.cbp` – Code::Blocks project file
- `lab1task.depend` – dependency file
